</tr>
</td>

# Deployment

Project is deployed on Heroku:
[Invoice-on-rails](https://invoice-on-rails.herokuapp.com/)

# Project setup

```
$ bundle install
$ rails db:setup
$ rails server
```

# Testing

To run the test suite, run:

```
$ rspec
```

# Built with

- ruby -v: 2.5.3
- rails -v: 5.2.3
- postgreSQL
